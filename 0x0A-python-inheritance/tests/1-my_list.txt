The MyList class
================

Using MyList
------------

Import the class from the module:
    >>> MyList = __import__('1-my_list').MyList

Check the class type:
    >>> ml = MyList()
    >>> type(ml) == MyList
    True

Check if it's an instance of list:
    >>> isinstance(ml, list)
    True

Check if print_sorted is an instance method:
    >>> type(MyList.__dict__['print_sorted'])
    <class 'function'>

Test with an empty list:
    >>> ml = MyList()
    >>> ml.print_sorted()
    []

Test with a normal list:
    >>> ml = MyList([1, 4, 2, 3, 5])
    >>> ml.print_sorted()
    [1, 2, 3, 4, 5]

Test with negative integers:
    >>> ml = MyList([-1, 4, 100, -33, 0])
    >>> ml.print_sorted()
    [-33, -1, 0, 4, 100]
